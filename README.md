##### MSI-GE60-2PL-403XCN-EFI
适用于MSI GE60 2PL 403XCN型号笔记本的Clover引导文件

# 仍存在的问题：
* 摄像头无法工作
* 无线网卡无法驱动（Intel无线网卡通病）
* GTX独立显卡无法驱动（多数双显卡笔记本通病）
* 概率性开机后蓝牙不可用
* 快捷键无法调整亮度，但系统设置中可以
* 概率性开机后登录微信开始花屏，若没有及时退出微信则画面冻结，无法操作（重启恢复）

# 已经解决的问题：
* HD4600驱动后开机花屏的问题
* USB插口无法识别设备并提示需要电源的问题
* Clover启动条目过多的问题
* 安装VoodooHDA PKG并手动选择路径可以解决没有声音的问题
* 睡眠秒醒的问题
